📊 Amazon Sales Performance Dashboard
📌 Project Overview

This project analyzes Amazon sales data to uncover insights related to revenue trends, product performance, and customer behavior using an interactive Excel dashboard.

🎯 Objectives
Analyze monthly revenue trends
Identify top-performing products
Understand category-wise sales distribution
Evaluate impact of discounts on revenue
Analyze customer regions
🛠 Tools Used
Excel (Pivot Tables, Dashboard Design)
SQL (Data Cleaning & Transformation)
📈 Key Insights
Discounts do not always lead to higher revenue
Certain categories consistently outperform others
Wallet payments show higher transaction value
